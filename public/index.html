部署说明

这个仓库包含前端落地页和后端示例，前端通过 GitHub Pages 发布，后端可以本地运行用于演示数据。

git clone https://github.com/mollyway8/001.git
cd 001
# 前端自动部署在 main 分支上
# 本地后端：cd backend && npm install && npm start

后端演示

简单但实用的 Express API

GET /

返回后端服务启动状态。

GET /api/status

返回运行状态、时间戳与服务 uptime。

GET /api/quote

返回一条随机激励语录，适合前端展示调用效果。

请先运行后端服务，再点击获取实时状态。
后端启动：cd backend && npm install && npm start
前端部署：GitHub Actions 会自动将 `public/` 发布到 `gh-pages`。
